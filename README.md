
# Ad Recommendation & CTR Prediction System

This project demonstrates an end-to-end click-through-rate (CTR) prediction pipeline using:
- Python
- TensorFlow
- scikit-learn
- Pandas
- NumPy

## Features
- Synthetic advertising dataset generation
- Feature engineering for behavioral/contextual signals
- Logistic Regression baseline
- Gradient Boosting model
- Neural Network model with TensorFlow/Keras
- Offline evaluation metrics:
  - AUC
  - Log Loss
  - Precision/Recall compatible workflows
- Recommendation ranking example

## Run

Install dependencies:

```bash
pip install pandas numpy scikit-learn tensorflow
```

Run:

```bash
python train_ctr_model.py
```
